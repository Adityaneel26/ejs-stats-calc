# EJS Simple Stats Calculator

A small but real utility project built with Node.js, Express, and EJS.

## Features
- Takes a comma-separated string of numbers as input.
- Calculates the **Mean (Average)**.
- Calculates the **Median (Middle Value)**.
- Displays the results back to the user on the same page.

## How to Run
1. `npm install`
2. `node index.js`
3. Open `http://localhost:3000`
